{name}

Modifies an input signal based on a control signal, a 'tipping point' and a scale factor.
A typical application is a 'keyboard scaling' to make higher tones softer than lower tones.
(We'll take this example for the input/output descriptions below)

Inputs:
  • IN: the input signal (connect from a amplitude output, for example an amp ADSR-OUT or MIDI-velocity)
  • ctrl: the control input (connect from a frequency output, for example MIDI-freq)
  • center: the 'tipping' point that will always remain unmodified (for example the 'center C' frequency)
  • scale: the scaling factor (0=no scaling, >0=attenuation of frequencies above 'center', <0=amplification of frequencies above 'center')
Outputs:
  • OUT: the output (connect to the 'lvl' input of an Amp that controls output volume)
  • ctrl-o: the passthrough of the ctrl input (connect to the frequency input of an oscillator)
