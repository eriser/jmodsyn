{name}

Merge multiple frequency/velocity/trigger inputs to a single one, quickly alternating between the values of the inputs.

Inputs:
  • length: The speed of the arpeggio in number of samples
  • frq(x): A frequency input
  • vel(x): A velocity input
  • trg(x): A trigger input
Outputs:
  • frq: The output frequency
  • vel: The output velocity (which is the average of the input velocities)
  • trg: The trigger output
