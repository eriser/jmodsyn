{name}

A wave-table oscillator that prevents aliasing (except when using pwm).

Inputs:
  • freq: the frequency in Hz
  • pwm: pulse-with modulation in % (50 is no pwm)
  • det: detune factor (1=no detune, 2=one octave up, 0.5=one octave down)
  • shp: the shape (you can also use the scroll-wheel to select the shape)
Outputs:
  • OUT: the output
